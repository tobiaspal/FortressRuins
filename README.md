# Fortress Ruins — Factorio 2.1 port

A Factorio 2.1-compatible release of [FortressRuins](https://mods.factorio.com/mod/FortressRuins) by Sacredanarchy (MIT).

Only `factorio_version` was bumped to 2.1; the mod content is unchanged. It plugs into [Abandoned Ruins - Updated](https://mods.factorio.com/mod/AbandonedRuins_updated_fork) (the engine) on Factorio 2.1.
